## Key points

- nCoV 2019 health API
- Api keys & access tokens
- REST Client (`http` | `dio`)
- Parsing JSON (`json_serializable`)
- State management (`provider`)
- Caching with shared preferences (`shared_preferences`)
- localization (`intl`)
- Error handling
- Cross platform (android ios web)

  
## API Reference

nCoV 2019 API provides information related with the case of pneumonia associated with the COVID-19 coronavirus.
